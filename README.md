# Budget Tracker 
Ein einfacher Budget Tracker in Python zur Verwaltung von Einnahmen und Ausgaben. 

## Ziel das Projekts 
- Einnahmen hinzufügen 
- Ausgaben hinzufügen 
- Transaktionen löschen 
- Kontostand berechnen 
- Speicherung in einer JSON-Datei

## Projektstruktur
```text  
budget-tracker/
|__ data.json
|__ main.py
|__ storage.py
|__ tracker.json
|__ tracker.py
```
## Starten 
python main.py

## Verwendete Technologien 
- Python 
- JSON
- datetime
